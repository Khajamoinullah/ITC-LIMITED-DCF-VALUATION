# ITC-LIMITED-DCF-VALUATION
## Project Overview
A bottom-up Discounted Cash Flow (DCF) valuation of ITC Limited, one of India's 
largest FMCG and cigarette conglomerates (NSE: ITC), built entirely in Microsoft Excel.

## Valuation Summary
 Metric - Value 
Implied Share Price |-₹185.4 |
Current Market Price - ₹309.7 |
Upside / Downside - −40.1% |
*Conclusion* - *Overvalued* 

## Model Structure
The workbook contains 8 sheets:
- Teaser— One-page company profile and valuation summary
- WACC — Cost of equity (CAPM), cost of debt, and capital structure
- ADJ DATA — Adjusted stock and Nifty 50 price data for beta calculation
- Data Sheet — Raw financial data sourced from Screener.in
- Profit & Loss — 10-year P&L with forecasts (Best & Worst case)
- Quarters— Trailing quarterly financials
- Balance Sheet — 10-year balance sheet data
- Cash Flow — 10-year cash flow statement
